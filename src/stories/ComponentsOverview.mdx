# Components

The EventPipe component library, grouped by **function** (Shopify Polaris–style).
Every component carries the design tokens — Azure primary, Graphite neutrals, 4px
radius, 40px control height. Foundational tokens live in **Foundations**;
full-screen app scaffolds live in **App Chrome**.

## Actions
Triggers and overlays. — **Button · Button Group · Link · Dropdown · Menu · Popover · Modal · Dialog · Backdrop · Side Panel**

## Navigation
Moving through the app. — **Breadcrumbs · Tabs · Pagination · Stepper**

## Forms
Data entry — the label-above field system (every input composes into **Form Field**).
**Form Field · Input · Input Number · Input Payment · Input Date · Input File · Input Tags · Pin Input · Input Group · Text Field · Text Area · Search · URL Field · Autocomplete · Time Picker · Select · Multi-Select · Checkbox · Checkbox Tree · Radio Group · Switch · Choice Chips · Slider · Range · Transfer List · Date Picker · File Upload · Rating**

## Feedback & Status
Communicating state. — **Badge · Badge Group · Alert · Banner · Progress · Skeleton · Loading Indicator · Snackbar · Toast · Empty States**

## Layout & Structure
Structure and content surfaces. — **Box · Container · Grid · Stack · Divider · Table · Card · Card Headers · Accordion · Section Header · List · List Item · Info Grid**

## Media & Visuals
Imagery and symbols. — **Avatar · Icon · Thumbnail · Image List · Carousel**

## Typography & Content
Text-level content. — **Text · Heading · Paragraph · Chip · Tooltip**

---

**Where things live:** tokens → **Foundations**; reusable components (above) →
**Components**; the sidebar / app bar / page header scaffold → **App Chrome**;
sign-in flows → **Account**.
